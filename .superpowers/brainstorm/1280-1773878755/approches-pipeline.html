Comment raccourcir la section Pipeline ?

3 approches pour réduire la hauteur — clique sur celle qui t'attire

A

Accordéon

Chaque pipeline (Setting / Closing) est fermé par défaut. On voit juste le label + une ligne de résumé. Clic pour déplier les cartes phases + showcase.

APERÇU
⚡ PIPELINE SETTING Détection → Qualification → Conversion ▼
🤝 PIPELINE CLOSING Coaching → Discovery → Délégation ▼

+

  • Gain max de hauteur (~80%)
  • Le visiteur choisit ce qu'il veut voir
  • Familier (pattern connu)

−

  • Contenu caché = moins de SEO visible
  • Le showcase "Concrètement" est caché aussi
B

Tabs Setting / Closing

Deux onglets côte à côte. Setting affiché par défaut, clic sur Closing pour switcher. Un seul visible à la fois.

APERÇU
⚡ Pipeline Setting
🤝 Pipeline Closing
Détection
Qualification
Conversion

+

  • Réduit ~50% de hauteur
  • Navigation claire entre les deux
  • Le contenu actif reste bien visible

−

  • L'utilisateur ne voit pas Closing sauf s'il clique
  • Moins standard sur landing page B2B
C

Résumé compact + "Voir le détail"

Chaque pipeline montre une ligne résumé avec les 3 phases (titres seulement). Un bouton "Voir le détail" déplie les cartes complètes et le showcase.

APERÇU
⚡ PIPELINE SETTING
01-03 Détection
04-06 Qualification
07-09 Conversion
Voir le détail ↓
🤝 PIPELINE CLOSING
Coaching
Discovery
Délégation
Voir le détail ↓

+

  • Le visiteur voit toute la structure d'un coup
  • Peut approfondir ce qui l'intéresse
  • Bon compromis SEO (titres visibles)

−

  • Gain de hauteur moyen (~60%)
  • Deux clics pour tout voir
