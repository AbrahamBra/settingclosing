A — Accordion

Chaque étape = une ligne compacte (numéro + titre + tag). Clic pour déplier la description et les outils. Une seule ouverte à la fois.

Pour

  • Scroll réduit de 80%
  • Toute l'info reste accessible
  • Pattern UX familier

Contre

  • L'info est cachée par défaut
  • Demande des clics pour explorer

B — 3 Phases groupées

9 étapes regroupées en 3 blocs : Détection (01-03), Qualification (04-06), Conversion (07-09). Chaque bloc = résumé + outils. Les numéros restent visibles.

Pour

  • Scroll réduit de 70%
  • Narrative claire en 3 temps
  • Pas besoin de clic

Contre

  • Perd le détail par étape
  • Les descriptions sont fusionnées

C — Phases + Accordion (hybride)

3 phases cliquables qui dévoilent leurs étapes. Chaque étape est aussi un accordion. Deux niveaux de profondeur : phase → étape → détail.

Pour

  • Scroll réduit de 85%+
  • Zéro perte d'info
  • Structure narrative forte

Contre

  • 2 niveaux de clic max
  • Plus complexe à implémenter
